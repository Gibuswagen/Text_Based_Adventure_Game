# Text_based adventure game
by Ibragim Tolegenuly

# Game description:
Upon starting user needs to input a name. A player starts with a random amount of money used to buy items in the Shop.
The shop has weapons, keys to open treasure, armour to reduce damage received as well as heal through healing pads.
There are 4 stages with an enemy encounter where player can choose to fight and receive rewards upon winning.
The player's goal is to reach 10 Points. If player's points goes below 0, he loses.
